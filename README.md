# OBS Netease Music Now Playing

![Screenshot](screenshot.png)

This script can export some information of your Netease Music current track to OBS-Studio text source.

**!!!!! macOS ONLY !!!!!**

Avaliable data:

- Title
- Artists
- Alias
- Album
